# Esercizi — 03 Array

## Traccia
- Dato un array di prezzi, calcola prezzi ivati, filtra quelli > 12 e somma.

## Requisiti
- Vietato usare funzioni di ordine superiore: `array_map`, `array_filter`, `array_reduce`, `foreach`.
- Usa esclusivamente cicli tradizionali (`for`, `while`) e, dove utile, array associativi.
- Mantieni la stessa funzionalità dell'esempio originale.

## Suggerimenti
- Scomponi il problema in tre passi: trasformazione, filtro, somma.
- Per gli array associativi, itera sulle chiavi con `array_keys()` e un ciclo `for`.
- Commenta chiaramente ogni blocco di codice per spiegare la logica.
